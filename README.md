# Retail Sales Data Analysis using Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to identify customer purchasing behavior, product performance, and sales trends.

The analysis was performed using Python and data analysis libraries to transform raw data into meaningful business insights.

---

## Objectives

* Clean and preprocess retail sales data
* Identify top-performing product categories
* Analyze customer purchasing patterns
* Study monthly sales trends
* Generate business insights using visualizations

---

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Google Colab
* GitHub

---

## Dataset Information

The dataset contains retail transaction records including:

* Customer Information
* Product Categories
* Quantity Purchased
* Sales Amount
* Transaction Dates

---

## Data Cleaning Steps

* Checked missing values
* Removed duplicate records
* Converted date columns into datetime format
* Created additional time-based features

---

## Exploratory Data Analysis

The following analyses were performed:

### Revenue Analysis

* Total Revenue Generated
* Revenue by Product Category

### Customer Analysis

* Gender-wise Sales
* Top Customers

### Sales Trends

* Monthly Sales Performance
* Product Category Trends
